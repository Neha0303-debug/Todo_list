## 📝 To-Do List App

A simple and responsive To-Do List web application built using HTML and CSS. This project helps users organize their daily tasks efficiently.
## 🚀 Features

1. Add tasks (UI-based)
2. Mark tasks as completed (UI design)
3. Clean and minimal interface
4. Responsive design (works on mobile & desktop)

## 🛠️ Tech Stack

1. HTML5
2. CSS3

## 📂 Project Structure

* index.html
* style.css

## 💡 What I Learned

Structuring a webpage using HTML
Styling using CSS (flexbox, layout, spacing)
Designing user-friendly UI
